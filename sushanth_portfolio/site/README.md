# Sushanth Kasturi - Portfolio

React + Vite + Tailwind. Five pages (Home, About, Skills, Projects, Contact). Floating Print + Download PDF buttons render an ATS-clean single-page resume.

## Develop
```
npm install
npm run dev
```

## Build
```
npm run build
npm run preview
```

## Edit content
All text lives in `src/data.js`. Update there, everything (pages + resume sheet) reflows.
